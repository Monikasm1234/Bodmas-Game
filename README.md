# 🎯 BODMAS Master Pro

An interactive web-based game designed to help users learn and practice **BODMAS (Order of Operations)** in a fun and engaging way.

---

## 📌 About the Project

This game teaches users how to solve mathematical expressions step-by-step using the BODMAS rule:

**B → Brackets  
O → Orders  
D → Division  
M → Multiplication  
A → Addition  
S → Subtraction**

Users must select the correct operation at each step, reinforcing proper problem-solving logic.

---

## 🚀 Features

✨ Interactive gameplay  
✨ Step-by-step problem solving  
✨ Real-time feedback with explanation  
✨ Score tracking system  
✨ Timer for each question  
✨ 3 difficulty levels:
- 🟢 Easy (Addition & Subtraction)
- 🟡 Medium (Includes Multiplication & Division)
- 🔴 Hard (Includes Brackets)

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 🎮 How to Play

1. A mathematical expression is displayed  
2. Multiple operation options are shown  
3. Click the correct operation based on BODMAS  
4. Continue solving step-by-step  
5. Earn points for correct answers
